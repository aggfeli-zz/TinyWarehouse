# TinyWarehouse
Project 1 - Operating Systems 2015

Το Project 1 αποτελείται από τα εξής πηγαία αρχεία:twa.c double_list.c linear_hashing.c myRecordDef.c
καθώς και απο τα αρχεία επικεφαλίδας: double_list.h linear_hashing.h myRecordDef.h

Στα αρχεία myRecordDef.c-myRecordDef.h : Γίνεται η δήλωση του struct που αφορά τα στοιχεία κάθε πελάτη, η ανάθεση τιμών και μια συνάστηρη εκτύπωσης.

Στα αρχεία  double_list.c- double_list.h : Γίνεται η δήλωση και η δημιουργία της διπλά συνδεδεμένης λίστας όπου υπάρχουν οι κατάλληλες συναρτήσεις που υλοποιούν τις απαιτούμενες πράξεις που χρειάζονται.Οι συναρτήσεις αυτές κάνουν τις εξής λειτουργίες : δημιουργία λίστας,καταστροφή λίστας, έλεγχος άδειας λίστας, εισαγώγη, ενημέρωση περιεχομένου, εμφάνιση κ πρώτων και κ τελευταίων εγγραφών, μέσος όρος  καθώς και οι ζητούμενες range και percentile. Τα αρχεία αυτά έχουν βασιστεί σε κώδικα που χρησιμοποιήθηκε και για την επίλυση ασκήσεων του μαθήματος Δομές Δεδομένων. 

Στα αρχεία  linear_hashing.c- linear_hashing.h : Γίνεται η δήλωση και η δημιουργια του hashtable καθώς και η δημιουργία μιας απλά συνδεδεμένης λίστας που απαιτείται για τον γραμμικό κατακερματισμό.Οι συναρτήσεις που έχουν υλοποιηθεί είναι : δημιουργια  hashtable και απλά συνδ.λίστας, καταστροφή, εισαγωγή στοιχείου στο  hashtable, ανάζήτηση και μια συνάρτηση που διαχειρίζεται το split και φροντίζει να γίνουν τα απαραίτητα στοχεία rehash. Το  hashtable είναι ένας πίνακας που κάθε κελί δείχνει σε μια απλά συνδεδεμένη λίστα.

Στο αρχείο  twa.c :  Καλούνται οι συναρτήσεις των παραπάνω αρχείων για την υλοποίηση των διάφορων εντολών. Στην αρχή, ανοίγεται το αρχείο που έχει δοθεί σαν όρισμα (argv[2]) και διαβάζονται ένα-ένα τα δεδομένα τα οποία εισέρχονται στην διπλά συνδεδεμένη λίστα και στο  hashtable. Στην συνέχεια το πρόγραμμα περιμένει τις κατάλληλες εντολές για να τις υλοποιήσει μέσω των switch-case. Οι εντολές που μπόρούν να δοθούν είναι όπως ακριβώς έχει ζητηθεί εκτός απο τις περιπτώσεις των 6. a(verage) και 9. e(xit) όπου θα πρέπει να δοθεί ένας οποιοσδήποτε παραπάνω χαρακτήρας.
